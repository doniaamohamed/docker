CMD & ENTRYPOINT:
CMD is used to define the default command that runs when the container starts
ENTRYPOINT is used to make the container work like an executable application
CMD gives default arguments or commands while ENTRYPOINT defines the main command of the container

COPY & ADD:
COPY is used to copy files and folders from the local machine into the container
ADD does the same thing as COPY, but it also supports:downloading files from URLs,extracting compressed tar files automatically.
